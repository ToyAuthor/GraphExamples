# 《3D繪圖程式設計》
程式碼及資源出自2008年碁峯資訊出版的《3D繪圖程式設計》一書  
此修改發佈已徵求過此書作者彭國倫先生的同意  

### 主要修改如下
- 略過Direct3D相關部分，只編譯OpenGL
- Build system改以CMake搭配MinGW 或者 Visual Studio 2010
- 增加一個新區塊，用來寫跨平台的範例
